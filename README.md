## Features

- ✔︎ React server side render
- ✔︎ ES6/ES7 supported
- ✔︎ Css modules supported
- ✔︎ Lots of [plugins](https://github.com/search?q=topic%3Aegg-plugin&type=Repositories)

## How to use

```bash
npm install beidou-init -g
beidou init
npm run start
```

After that, go to `http://localhost:6001/`
