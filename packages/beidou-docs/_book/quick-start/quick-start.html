Quick Start

Quick Start

Requirements

  • Mac OS X, or Linux
  • Node.js v6.0 or newer

Setup

Install generator

npm install beidou-init -g

Init

beidou init

Start

npm install
npm run start

After that, go to http://localhost:6001/
